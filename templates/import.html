Supported Formats

CSV Files

Most bank CSV exports work automatically. The importer detects columns named:

  • Date: date, posted, transaction date
  • Description: description, memo, narrative, details, payee
  • Amount: amount, value, or separate debit/credit columns

PDF Files

PDF parsing extracts transactions from tables. Results may vary. For best results, export as CSV from your bank when possible.

Duplicate Protection

The importer tracks file hashes to prevent importing the same file twice.
