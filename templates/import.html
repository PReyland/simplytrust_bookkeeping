Supported Formats

CSV Files

Most bank CSV exports work automatically. The importer looks for columns named:

  • Date: date, posted, transaction date
  • Description: description, memo, narrative, details, payee
  • Amount: amount, value, or separate debit/credit columns

PDF Files

PDF parsing attempts to extract transactions from tables. Results may vary depending on the PDF format. For best results, export as CSV from your bank if possible.

Duplicate Protection

The importer tracks file hashes to prevent importing the same file twice.
